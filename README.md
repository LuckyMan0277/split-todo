# Split Todo

> 할 일을 세분화하여 시작의 장벽을 낮추는 체크리스트 기반 태스크 관리 앱

## 📖 프로젝트 모티브

할 일 관리가 제대로 되지 않아 마감을 넘기거나, 하루 종일 아무것도 하지 못하고 지나가는 경험을 해본 적이 있으신가요?

ADHD를 가진 사람들을 포함해 많은 사람들이 **큰 작업을 시작하는 것 자체에 어려움**을 느낍니다.
이 앱은 이러한 문제를 해결하기 위해 만들어졌습니다.

**핵심 아이디어**: 할 일을 작은 세부 목표로 쪼개고, 그것들을 하나씩 체크해 나가면 **시작에 필요한 에너지를 크게 줄일 수 있습니다.**

## ✨ 주요 기능

### 📋 체크리스트 기반 할 일 관리

- 큰 작업을 작은 **세부 할 일**로 나누어 관리
- 체크리스트 형태로 하나씩 완료하는 성취감

### 📅 오늘 할 일 페이지

- 일자별로 할 일을 확인하고 관리
- 오늘에 집중할 수 있는 인터페이스

### ✅ 간편한 체크리스트 인터페이스

- 직관적인 UI로 빠르게 할 일 추가 및 완료 체크
- 진행 상황을 한눈에 확인

- **React Native** (Expo)

